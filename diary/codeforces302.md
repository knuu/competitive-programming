# div1 A. Writing Code

- 問題
    - N人で合計M行のコードを書くとする。i番目の人が1行書くごとにa[i]個のバグを出すとき、バグをB個以下に抑えるような行数の割り当て方は何通りあるか？
- 解法
    - DP
    - dp\[r\]\[b\] = (r行目まで書き終え、b個バグが発生したときの場合の数)
    - テーブルを0で初期化し、dp\[0\]\[0\] = 1
    - 更新は dp\[r+1\]\[b\] += dp\[r\]\[b-a\[i\] (i:0->N-1, r:0->R-1, b:a[i]->B)
    - 答えはsum{dp[M] | M in [0, B]}
    - O(NMB)
- タグ: DP
- レベル: 2
